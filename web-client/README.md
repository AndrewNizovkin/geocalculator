# web-client

Включает в себя клиентскую часть приложения, работающую в браузере пользователя. 
Предоставляет графический интерфейс для решения прикладных задач геодезии:

- Каталог координат. Предоставляет возможность создавать, читать из текстового файла,
редактировать базу данных трёхмерных координат точек, которая может быть использована 
при решении других задач в системе.

- Топографическая съёмка. Предоставляет инструменты для обработки тахеометрической съёмки. 
Исходные данные могут вводиться вручную или импортироваться из файлов электронных тахеометров. 
Отображает, полученные в результате математической обработки каталоги и ведомости координат, 
схему измерений. 


Используемые технологии:

- HTML

- CSS

- JavaScript
